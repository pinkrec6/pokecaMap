# ポケカイベントマップ 🗺️

[ポケモンカードゲーム プレイヤーズクラブ](https://players.pokemon-card.com/event/search) のイベント検索結果を、地図上で見られるようにしたWebアプリです。サーバー不要の1枚のHTMLファイル（`index.html`）だけで動きます。

## 機能

- イベント開催店舗を地図上にピン表示（近いピンは自動でまとめて表示）
- ピンをタップすると、その会場のイベント一覧・参加費・定員・公式詳細ページへのリンクを表示
- 絞り込み: キーワード / 都道府県 / 開催期間 / イベント種類（シティリーグ、ジムイベントなど）
- 📋 リスト表示（タップで地図の該当ピンへジャンプ）
- 📍 現在地へ移動ボタン
- スマホ対応（レスポンシブ）

## 使い方

### PCで見る

`index.html` をダブルクリックしてブラウザで開くだけです。

### スマホで見る（同じWi-Fi内）

1. PCで `start-server.bat` をダブルクリック（Pythonが必要です）
2. 画面に表示されるPCのIPアドレスを確認（例: `192.168.1.10`）
3. スマホのブラウザで `http://192.168.1.10:8787/` を開く

※ Windowsファイアウォールの許可ダイアログが出たら「アクセスを許可」してください。
※ `http://` 接続のため、スマホでは📍現在地ボタンが使えない場合があります（ブラウザの制限）。

### どこからでもスマホで見る（おすすめ）

GitHub Pages や Netlify Drop などの無料ホスティングに `index.html` を置くと、URLひとつでどこからでも見られます（HTTPSになるので現在地ボタンも使えます）。

- [Netlify Drop](https://app.netlify.com/drop): ブラウザにフォルダをドラッグ&ドロップするだけ
- GitHub Pages: リポジトリにpushして Settings → Pages で公開

## 仕組み

- イベントデータ: プレイヤーズクラブの検索API（`/event_search`）をブラウザから直接取得
- 位置情報: イベントの住所を[国土地理院の住所検索API](https://msearch.gsi.go.jp/address-search/AddressSearch)で緯度経度に変換（結果はブラウザの `localStorage` にキャッシュされるので、2回目以降は高速）
- 地図: [Leaflet](https://leafletjs.com/) + [OpenStreetMap](https://www.openstreetmap.org/)

## 注意

- 一度に読み込むのは300件までです。それ以上ある場合は地図上部の「さらに読み込む」を押してください。
- 住所から位置を特定できなかった会場は地図に表示されません（リストには表示されます）。
- 非公式ツールです。イベントの正確な情報は必ず公式ページで確認してください。
